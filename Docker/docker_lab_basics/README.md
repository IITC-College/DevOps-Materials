Learn the fundamentals of Docker by running and managing containers using popular images. Master essential Docker commands through hands-on practice with web servers and databases.

## Learning Objectives

After this lab you will be able to:
- Pull Docker images from Docker Hub
- Run containers from images
- List running and stopped containers
- View container logs
- Stop and remove containers
- Run containers with port mapping
- Run containers in detached mode
- Execute commands inside running containers
- Understand basic container networking

## Duration

30-40 minutes

## Prerequisites

- Linux system with Docker installed
- Docker daemon running
- Basic command line knowledge
- Internet connection to pull images

## What You'll Work With

Popular Docker images for hands-on practice:
1. **nginx** - A lightweight web server
2. **postgres** - A powerful database server
3. **hello-world** - Docker's test image

Each task teaches you essential Docker commands and concepts through practical examples.
