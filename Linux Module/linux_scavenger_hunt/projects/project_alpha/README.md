# Project Alpha

## Description
Main web application backend service

## Technology Stack
- Python 3.11
- Flask Framework
- PostgreSQL Database
- Redis Cache

## Setup Instructions
1. Install dependencies: pip install -r requirements.txt
2. Configure database: cp config.example.yaml config.yaml
3. Run migrations: python manage.py migrate
4. Start server: python app.py

## Configuration
See config.yaml for all configuration options.
