üìù What each command does
========================================================

1. Downloads the lab archive from GitHub
2. Extracts all files
3. Navigates into the lab directory
4. Displays the starting instructions (start_here.txt or clue1.txt)

========================================================

üí° Tips
========================================================

- Ensure curl and tar are installed
- Run the command in the directory where you want the lab extracted
- Labs are self-contained and can be done in any order
- Use start_here.txt after extraction (or clues/level1/clue1.txt for clue-based labs)
