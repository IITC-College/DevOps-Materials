================================================================================
    SECURITY ANALYSIS
================================================================================

For each umask value, analyze:

**umask 022** (default_app/):
  - Who can read the files? (owner, group, others)
  - Who can write the files? (owner, group, others)
  - Security level: Low / Medium / High
  - When to use: Public files, shared resources, less sensitive data
  - Risk: Others can read your files (information leakage)

**umask 027** (shared_team/):
  - Who can read the files? (owner, group, others)
  - Who can write the files? (owner, group, others)
  - Security level: Low / Medium / High
  - When to use: Team projects, collaborative work, group access needed
  - Risk: Group members can modify files (need to trust group)

**umask 077** (secure_files/):
  - Who can read the files? (owner, group, others)
  - Who can write the files? (owner, group, others)
  - Security level: Low / Medium / High
  - When to use: Private files, sensitive data, personal documents
  - Risk: No sharing possible (collaboration blocked)
