# Script Functions Lab - Optional Tips

## Level 1 Clue 1 (check_dir, $1)
- check_dir() { if [ -d "$1" ]; then echo "Directory $1 exists"; else echo "Directory $1 missing"; fi }
- Call: check_dir logs or check_dir ../data/environments/project1 (from projects/).
- Quote $1 in [ -d "$1" ] in case the path has spaces.

## Level 1 Clue 2 (deploy_tier)
- deploy_tier() { echo "Deploying $1..."; echo "  $1: deployed"; }
- Call three times: deploy_tier frontend; deploy_tier api; deploy_tier worker.

## Level 1 Clue 3 (log_header)
- log_header() { echo "[$(date '+%H:%M:%S')] $1"; } – one place to change timestamp format.

## Level 2 Clue 1 (loop + function)
- for TIER in frontend api worker; do deploy_tier "$TIER"; done – $1 inside the function is the current TIER.

## Level 2 Clue 2 (function with two steps)
- deploy_tier() { echo "  Stopping $1..."; echo "  Starting $1..."; echo "  $1: deployed"; } – no duplication when called from loop.

## Level 2 Clue 3 (deploy_tiers script)
- One function deploy_tier, one for loop over tiers, optional log_header at start/end.
