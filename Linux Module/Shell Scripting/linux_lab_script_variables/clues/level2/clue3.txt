WHAT YOU'RE DOING

Combine what you learned: variables for app name, version, date, and backup
filename; echo statements for clear user messages. A good backup script
tells the user what it's doing at each step.

--------------------------------------------------------------------------------
INSTRUCTIONS
--------------------------------------------------------------------------------
1. Create a script in projects/: backup_full.sh
   Shebang, then variables: APP_NAME="webapp", VERSION="1.0",
   DATE=$(date +%Y-%m-%d), BACKUP_NAME="backup_${APP_NAME}_${DATE}.log"

2. Add echo statements that use these variables:
   - "Backup started for $APP_NAME version $VERSION"
   - "Date: $DATE"
   - "Backup file: $BACKUP_NAME"
   - "Backing up..."
   - "Backup complete."
