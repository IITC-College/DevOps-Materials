# pip install - Installing External Packages

## Basic Installation

# Install a package
pip install requests

# Install specific version
pip install requests==2.28.2

# Install minimum version
pip install requests>=2.28.0

# Install latest version (upgrade)
pip install --upgrade requests

## Multiple Packages

# Install multiple packages at once
pip install requests flask pytest

# Install from requirements.txt
pip install -r requirements.txt

## Uninstalling

# Uninstall a package
pip uninstall requests

# Uninstall without confirmation
pip uninstall -y requests

## Checking Installation

# List installed packages
pip list

# Show specific package info
pip show requests

# Check outdated packages
pip list --outdated

## Common Packages to Install

# Web Development
pip install flask
pip install django
pip install fastapi

# Data Science
pip install pandas
pip install numpy
pip install matplotlib

# Testing
pip install pytest
pip install unittest

# HTTP Requests
pip install requests

# Database
pip install psycopg2-binary  # PostgreSQL
pip install pymongo  # MongoDB

# CLI Tools
pip install click
pip install argparse

# Date/Time
pip install python-dateutil

## Best Practices

1. Always use virtual environment
2. Install one package at a time initially
3. Test after each installation
4. Update requirements.txt after installing
5. Use specific versions in production
