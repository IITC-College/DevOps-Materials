# Processes

Labs covering Linux process management, systemd services, and log inspection.

## Labs

| Lab | Description |
|-----|-------------|
| `linux_lab_processes_basics` | Viewing running processes with `ps`, `top`, and `htop` |
| `linux_lab_processes_kill` | Sending signals to processes with `kill`, `pkill`, and `killall` |
| `linux_lab_processes_vs_service` | Understanding the difference between a process and a managed service |
| `linux_lab_systemd_services` | Managing services with `systemctl` (start, stop, enable, status) |
| `linux_lab_journalctl_logs_first` | Reading system logs with `journalctl` |
| `linux_lab_troubleshooting_flow` | End-to-end troubleshooting flow for a broken service |

Each lab folder contains a `README.md` with full instructions.
