# Shell Scripting

Labs covering Bash scripting from basics to real-world automation scripts.

## Labs

| Lab | Description |
|-----|-------------|
| `linux_lab_script_basics` | Script structure, shebang line, making scripts executable |
| `linux_lab_script_variables` | Declaring and using variables in Bash |
| `linux_lab_script_input` | Reading user input with `read` |
| `linux_lab_script_conditionals` | `if`/`elif`/`else` conditionals in Bash |
| `linux_lab_script_case` | `case` statements for branching logic |
| `linux_lab_script_loops` | `for` and `while` loops |
| `linux_lab_script_exit_codes` | Using exit codes and `$?` for error handling |
| `linux_lab_script_functions` | Defining and calling functions in Bash |
| `linux_lab_script_debugging` | Debugging scripts with `set -x`, `set -e`, and `shellcheck` |
| `linux_lab_script_leap_year` | Logic exercise: detect leap years |
| `linux_lab_script_guessing_game` | Fun project: build a number-guessing game |
| `linux_lab_script_ops_helper` | Practical script: an operations helper menu |
| `linux_lab_script_mini_devops_tool` | Capstone: a small DevOps automation tool |

Each lab folder contains a `README.md` with full instructions.
