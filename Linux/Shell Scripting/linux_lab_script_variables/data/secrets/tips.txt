# Script Variables Lab - Optional Tips

## Level 1 Clue 1 (Hardcoded values)
- Use shebang and echo. No variables yet – just fixed text.

## Level 1 Clue 2 (Variables)
- No spaces: APP_NAME="myapp" not APP_NAME = "myapp"
- Use $APP_NAME in echo. Double quotes allow variable expansion.

## Level 1 Clue 3 (Variables in output)
- Single quotes prevent expansion: echo '$APP_NAME' prints literal $APP_NAME
- Space in assignment: APP_NAME = "myapp" makes bash try to run "APP_NAME" as a command – error.

## Level 2 Clue 1 (Command substitution)
- DATE=$(date +%Y-%m-%d) runs the date command and stores output in DATE
- From lab root, data/logs/ holds the dummy logs after setup.

## Level 2 Clue 2 (Timestamped filenames)
- BACKUP_FILE="backup_${DATE}.log" – curly braces separate variable from .log

## Level 2 Clue 3 (Complete script)
- Define APP_NAME and VERSION at top; change once to "backup a different app".
