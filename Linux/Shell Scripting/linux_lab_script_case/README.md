# Linux Lab – Script Case Statements

Master the bash `case` statement for building clean, maintainable menu-driven scripts. Progress from simple menus to ops tools that run real commands (no functions—those are in the Functions lab).

## Learning Objectives

After this lab you will be able to:
- Write case statements for menu systems
- Pattern match with numbers and strings
- Use the default branch `*)` for invalid input
- Exit with appropriate error codes
- Execute real commands in case branches

## Getting Started

Open `TASKS.md` to begin the lab exercises.

## Duration

20-30 minutes

## Prerequisites

- Script Basics, Variables, Input, and Conditionals labs
- Linux system with bash
- A text editor
