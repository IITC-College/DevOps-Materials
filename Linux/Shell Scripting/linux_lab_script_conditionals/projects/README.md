# Projects Folder

Create your validation and deployment scripts here as instructed in the clues.

- **check_config.sh** – Level 1 Clue 1 (file exists)
- **check_dir.sh** or **check_config.sh** – Level 1 Clue 2 (directory exists)
- **validate_input.sh** – Level 1 Clue 3 (empty input)
- **validate_env.sh** or **validate_input.sh** – Level 2 Clue 1 (prod/staging only)
- **ensure_dir.sh** – Level 2 Clue 2 (mkdir if missing)
- **pre_deploy_validate.sh** – Level 2 Clue 3 (full validation)
