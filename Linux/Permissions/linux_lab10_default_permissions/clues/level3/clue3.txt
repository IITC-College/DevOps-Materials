================================================================================
    THE CHALLENGE
================================================================================

Requirement: "I need to create files that are readable by my group members,
but not by others. Only I should be able to write to them."

Your task:
1. Determine what permissions the files should have
2. Calculate what umask value will produce those permissions
3. Set the umask
4. Create a test file
5. Verify the permissions match the requirement

================================================================================
    STEP-BY-STEP SOLUTION
================================================================================

Step 1: Determine required permissions
  - Owner: read + write = rw- (6)
  - Group: read only = r-- (4)
  - Others: nothing = --- (0)
  - Result: 640 (rw-r-----)

Step 2: Calculate umask
  - For files: 666 - umask = 640
  - Therefore: umask = 666 - 640 = 026
  - Wait... let's verify: 666 - 026 = 640 âœ“

Step 3: Set umask and test
  Run: umask 026
  Run: touch final_test.txt
  Run: ls -l final_test.txt
  Expected: -rw-r----- (640)

Step 4: Verify it meets the requirement
  - Owner can read and write? Yes / No
  - Group can read? Yes / No
  - Group can write? Yes / No (should be No)
  - Others can read? Yes / No (should be No)
