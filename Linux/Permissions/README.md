# Permissions

Labs covering Linux file permissions, ownership, groups, sudo, and default permission masks.

## Labs

| Lab | Description |
|-----|-------------|
| `linux_lab6_permission_encounter` | Reading and modifying permissions with `chmod` (symbolic and octal) |
| `linux_lab7_why_script_doesnt_run` | Diagnosing and fixing execute permission issues on scripts |
| `linux_lab8_ownership_and_groups` | Changing file ownership and group membership with `chown` and `chgrp` |
| `linux_lab9_sudo_mindset` | Understanding when and how to use `sudo` safely |
| `linux_lab10_default_permissions` | Controlling default permissions with `umask` |
| `linux_lab11_debug_mindset` | Debugging mixed permission scenarios across users and groups |

Each lab folder contains a `README.md` with full instructions.
